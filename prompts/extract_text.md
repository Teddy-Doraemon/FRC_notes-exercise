# extract_text.md

你现在要处理一组同一单元的法语教材截图。

任务：识别并整理所有截图中的原始教材内容，输出为 **markdown 正文**，供后续教学笔记生成使用。

请严格遵守以下要求：

1. 按系统提供的图片顺序整理内容。
2. 只提取图片中真实可见的信息，不要补写图片中没有的内容。
3. 不要翻译法语原文；如果图片里本来就有中文解释或中文说明，要原样保留。
4. 尽量保留教材顺序，包括：
   - 单元标题
   - 课文正文
   - 听力材料
   - 对话
   - 词汇表
   - 语法点
   - 教材中出现的练习说明
   - 中文说明
5. 如果局部无法辨认，请明确标注 `【图片不清晰】`，不要猜测。
6. 输出必须适合作为 `output/complete_texts.md` 直接保存。

推荐输出结构：

```md
# 教材原文整理

## 单元标题

## 课文 / 对话 / 听力材料

## 词汇与语法

## 练习说明
```

如果教材实际结构不同，可以按实际内容调整，但仍要保持清晰、完整、顺序准确。
